Build a production-quality website using typescript and Tailwind CSS.

the project is a version of the website for schools, Niche, linked here, but for study abroad websites.
https://www.niche.com/

Requirements:

* Clean, modern, minimal design, 
* Fully responsive
* Strong typography and spacing
* Reusable components
* Accessible semantic HTML
* No unnecessary dependencies
* Keep code organized and easy to extend

Process rules:

* First, explain the implementation plan
* Then list the files you will create or modify
* Make changes in small, logical steps
* After each major step, summarize what changed and how to test it
* Do not add backend/auth/database unless explicitly requested

Initial scope:

* Homepage only
* Navbar
* Features section
* Footer

Design direction:

* Professional, sharp, high-end feel
* Minimal but not empty
* Good mobile layout
